Projeto de detecção de fraudes utilizando aprendizado supervisionado.
 
Funcionalidades

- Interface web interativa (Streamlit)
- Pré-processamento de dados (feature engineering, padronização)
- Treinamento com RandomForest + SMOTE para balanceamento de classes
- Filtros e dashboards para análise das transações
- Feedback humano para re-treinamento do modelo
- Visualização de métricas (Falsos positivos/negativos, ROC, PR, boxplot, histogramas)

Como executar

1# Instale os requisitos (veja o requirements.txt).
2# Baixe o arquivo de dados (CSV).
3# Execute o aplicativo:
	streamlit run analise_fraudes.py

Dependências

- Veja o arquivo requirements.txt para detalhes.
